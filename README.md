# Mini AI Agent

Simple educational AI agent that can:

- Answer questions
- Use a calculator tool

Run:
#Clone the repository:

git clone https://github.com/AwsAlkhateeb/mini-ai-agent.git

#Setup your OpenAI API key:

Create a file named .env in the project root.

Add your own OpenAI API key like this:

OPENAI_API_KEY=YOUR_API_KEY

#Install dependencies:

pip install -r requirements.txt

#Run the Agent:
python main.py


#Type your questions or math expressions. To exit, type:

exit
